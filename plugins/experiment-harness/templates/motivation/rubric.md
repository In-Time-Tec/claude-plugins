# Rubric

Each dimension is scored **0–10** per arm, with 0 = absent/broken and 10 = exemplary. Score honestly from the artifact alone; do not speculate about how the artifact came to be.

For every dimension, your justification must cite **specific evidence** from the arm's files (quoted HTML/CSS, file paths, line numbers).

## Dimensions

### 1. Semantic HTML

Are semantic elements (`<header>`, `<main>`, `<section>`, `<article>`, `<nav>`, `<footer>`) used where appropriate? Is the document outline sensible? Are heading levels correctly nested?

### 2. Accessibility

Are all images accompanied by meaningful (or deliberately empty) `alt` text? Are interactive elements keyboard-reachable and have visible focus states? Are form fields properly labeled? Is color contrast plausibly sufficient (judge from the CSS values)? Any ARIA misuse?

### 3. Visual design

Color palette coherence. Typography: font choice, hierarchy, line length, line height. Spacing rhythm. Alignment. Overall composition. Does the page feel like someone chose these things on purpose?

### 4. Code quality

CSS organization (grouped sensibly, reasonable specificity, no obvious duplication). HTML cleanliness (readable indentation, no stray inline styles, no obvious hacks). Any signs of unused code.

### 5. Creative distinctiveness

How much does this feel like *this specific premise*, rather than a generic landing page template? Do design and copy choices reflect the club's mood? Are there any small moments of genuine craft?

### 6. Content quality

Is the written copy plausible, coherent, and specific to the club's premise? Is it the right amount — not thin, not overwrought? Does it communicate the offer clearly within the 30-second read the task asks for?

### 7. Polish

Microtypography (widows, hanging punctuation, em-dashes where appropriate). Responsive behavior (judged from CSS — media queries, flexible widths). Hover/focus states. The presence or absence of small thoughtful details.

## Scoring

Produce `review.md` following the format specified in the reviewer skill. For each dimension give:

- A score for arm A and arm B.
- A 1–3 sentence justification per arm citing specific evidence.

Then give a single overall winner (A, B, or tie) with a paragraph of rationale.
